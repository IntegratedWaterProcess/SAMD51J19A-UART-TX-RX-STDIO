# SAMD51J19A-UART-TX-RX-STDIO
SAMD51J19A Chip on a Adafruit Metro M4 Development Board w/ Baremetal C Drivers.
CPU clocked at 119,997,440 Hz (120 MHz)
This application transmits "Hello Dude!" via Serial Communications Port 5 (SERCOM 5) that is configured as UART. The to the console using an FT232R TTL serial converter.
TX PIN -> PB02 SERCOM5 PAD 0
RX PIN <- PB03 SERCOM5 PAD 1
